从登录框组件偷窥flex布局

发表于 2017-06-18

前言

众所周知，前端技术更新换代的速度，绝不是一般的技术能够媲美的。各种框架的推出，也为我们提供了各种各样的轮子，可以尽情玩耍。但是随着学习的展开，我越来越意识到一个问题：没有基础是走不远的。只会调调API的程序员，终究只能是搬砖的，很难成长为高级工程师。因此，笨人下笨功夫，我打算着手开始一个造轮子系列，把各种常用的轮子都造一遍。

登录框组件

结构

  • 外层 div 包裹 form 表单；
  • 表单中放置两个 input 输入框；
  • i 标签用于引入图标；
  • a 标签链接更改密码或忘记密码；
  • button 按钮提交表单

实现

实验在codepen中完成。纯css打造的登录框组件。暂未实现交互效果。戳我看源码

知识点：

  1. 外层div 垂直水平居中；
  2. transition 和 transform 实现动画效果；
  3. scss 语法；
  4. 盒模型；
  5. css选择器知识；
  6. flex、绝对定位等布局知识。

flex布局

我是参考阮一峰老师关于弹性布局讲解的一篇文章来学习的。这里单独拎出来写一下，是为了加深映像。

  • 将最外层包裹元素指定为弹性布局

  • flex-direction 指定主轴的排列方向

  • flex-wrap 指定是否换行

  • flex-flow
    flex-direction 属性与 flex-wrap 的简写 默认为： row nowrap

  • justify-content 属性定义了在主轴上的对齐方式

  • align-items 属性定义了在交叉轴上的对齐方式

  • align-content 多跟轴线的对齐方式 若只有一根轴线 不起作用
